--- layout: post title: Opening Files in TextMate Project View through Cyberduck date: 2011-10-29 05:32:27 UTC updated: 2011-10-29 05:32:27 UTC comments: false categories: ftp cyberduck code textmate --- At work I edit a lot of files on a remote development server with TextMate using Cyberduck.  When having to edit multiple files in the same directory it can get a bit confusing because using the edit with TextMate feature opens each file in a separate window.  To get around this and have TextMate open the files in the project view I did the following.
  1. Make sure you have Cyberduck set up to use TextMate.

  2. In Cyberduck select the file you want to edit and click the edit with TextMate button.

  3. In the window that opens hit control + shift + o (a terminal will open)

  4. In the terminal that opens issue the command "mate ."

  5. Now every file that you open through TextMate will open as a tab in the project view.
