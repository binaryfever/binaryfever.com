--- layout: post title: Ruby Net::SSH date: 2012-09-29 04:40:21 UTC updated: 2012-09-29 04:40:21 UTC comments: false categories: ruby code --- After meeting up with some old CS classmates at Google IO a few days ago I decided to try and login to one of the old servers we used for classes way back when. No luck, it seems the Sys. Admin purges accounts after six months of inactivity and I graduated in 2008... so yeah.  After I shot him a quick email he was nice enough to create a new account for me.  Knowing myself though, I will not use it for six months but then find a use for it on the seventh month.  So I decided to write a quick ruby script that I set up with a cron job which will run on another server I have.  With the net-ssh gem it was easy.

*thanks to twothirtyam for helping correct my beer induced spelling and grammer errors*


 require 'rubygems'  
 require 'net/ssh'  
 server = 'yourserver'  
 username = 'yourusername'  
 password = 'yourpassword'  
 Net::SSH.start(server, username, :password => password) do |ssh|  
  ls_result = ssh.exec("ls -l")  
  puts ls_result  
 end
