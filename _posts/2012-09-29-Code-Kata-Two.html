The first Code Kata is dealing with Binary Searches on a sorted array.  Pretty standard stuff when it comes to the iterative and recursive approaches. Coming up with three other implementations is going to be fun!

Here are my solutions for the iterative and recursive approach.  I'll add to the Binarychop class as I come up with other approaches.

 #binarychop.rb  
 class Binarychop  
  def iterative(n, array)  
   min = 0  
   max = array.size  
   while min < max  
    mid = (min + max) / 2  
    if n > array[mid]  
     min = mid + 1  
    elsif n < array[mid]  
     max = mid - 1  
    else  
     return mid  
    end  
   end  
   return -1  
  end  
  def recursive(n, array, min, max)  
   return -1 if max < min  
   mid = (min + max) / 2  
   if n < array[mid]  
    recursive(n, array, min, mid - 1)  
   elsif n > array[mid]  
    recursive(n, array, mid + 1, max)  
   else  
    return mid  
   end  
  end  
 end

and the rspec tests just because it is my first time using rspec outside of rails.

 #binarychop_spec.rb  
 require 'binarychop'  
 require File.dirname(__FILE__) + '/helper'  
 require File.dirname(__FILE__) + '/../binarychop.rb'  
 describe Binarychop, "#iterative" do  
  it "should return -1 if number not found" do  
   binarychop = Binarychop.new  
   binarychop.iterative(3, [1, 2, 4]).should == -1  
  end  
  it "should return an integer if the number is found" do   
   binarychop = Binarychop.new  
   binarychop.iterative(3, [2, 3, 4]).should_not == -1  
  end  
 end  
 describe Binarychop, "#recursive" do  
  it "should return -1 if max is less then min" do  
   binarychop = Binarychop.new  
   binarychop.recursive(3, [1, 2, 4], 0, -1).should == -1  
  end  
  it "should return an integer if the number is found" do  
   binarychop = Binarychop.new  
   binarychop.recursive(3, [1, 2, 3], 0, 3).should_not == -1  
  end  
 end
